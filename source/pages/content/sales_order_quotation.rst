Sales Quotation / Penawaran
===========================

Untuk dapat menginput penawaran dapat dilakukan dengan menu "Sales" -> "Sales" -> "RFQ".
Ketika klik menu RFQ maka user akan diarahkan pada halaman RFQ. Seperti gambar dibawah.
